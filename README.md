## 简介
PigchaVPN是一款用于多平台系统的科学上网，翻墙，上网加速，VPN工具，上网代理，浏览器代理，Git代理，控制台代理的客户端。如果本项目对您有用，请不要吝啬你的star~

- mac
- ubuntu
- ios 开发中
- android 开发中

## 注意事项
- 请关闭其它同类软件
- 关闭浏览器相关代理插件
- 请关闭杀毒相关软件，承诺本软件无任何病毒
